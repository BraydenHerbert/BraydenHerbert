Projects

Hardware-in-the-Loop Power Supply PCB

Designed a 4-layer PCB for powering hardware-in-the-loop testing, managing power distribution across six vehicle systems. Integrated voltage regulators, logic switches, and current sensors controlled by an ESP32 using Altium and LTspice.

6502-based 8-Bit Computer

Built a functional 8-bit computer with the 6502 processor, including assembly programming and hardware design based on Ben Eater's design. Integrated circuits for memory management, clock generation, and address decoding.

Smart Medication Monitoring PCB

Designed a non-contact liquid-level monitoring device using a 4-channel I2C capacitance-to-digital conversion system, based on an ESP32 MCU with USB-C charging and an integrated web server for remote monitoring.

Model Wing with Electrical Control System

PWM-based control circuitry and firmware to adjust a linear actuator on a 1:5 scale F1 wing.
